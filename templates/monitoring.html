Apple Quality Model Monitoring

This page provides monitoring information and links to the Prometheus metrics for the Apple Quality Prediction application.

The Prometheus metrics are automatically collected by the prometheus-flask-exporter and can be accessed at the /metrics endpoint.

Monitored Metrics

  • app_info - Basic application information including version
  • prediction_requests_total - Total number of prediction requests made
  • prediction_latency_seconds - Time taken to process prediction requests
  • last_prediction_value - Value of the last prediction made (0-1)
  • flask_http_request_total - HTTP request count by status code and method
  • flask_http_request_duration_seconds - HTTP request latency by endpoint

How to Connect to Prometheus

To view these metrics in Prometheus:

  1. Make sure your Prometheus server is configured to scrape this application
  2. The scrape configuration should include the target apple-production-kstarid:8080
  3. The metrics path should be set to /metrics
